# ExplainX – Explainable AI for Deep Learning Models

ExplainX is a Python and PyTorch based framework for interpreting machine learning and deep learning model predictions using SHAP, LIME and Grad-CAM.

## Features
- Train PyTorch model
- SHAP explanations
- LIME explanations
- Feature importance visualization
- Modular project structure
- Experiment pipeline

## Tech Stack
- Python
- PyTorch
- SHAP
- LIME
- Scikit-learn
- Matplotlib

## Setup
pip install -r requirements.txt
python main.py

explainers/
